# Task-App

## Tech Stack

- **Framework**: Nextjs 16 (React) - App Router
- **Language**: TypeScript
- **Database**: Supabase DB
- **Styling**: Tailwind CSS
- **Auth** : Supabase Auth
- **Deployment** : Vercel



## Project Structure
